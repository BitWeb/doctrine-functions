Doctrine-functions

Set of DQL functions for Doctrine 2

View project onGitHub

Doctrine Functions

This package contains doctrine functions, you can contribute by forking it and propose pull request with your own functions. By defaults it add these:

  • DATE(expr)
  • DATEDIFF(expr1, expr2)
  • DAYOFWEEK(expr)
  • DAYOFMONTH(expr)
  • DAYOFYEAR(expr)
  • HOUR(expr)
  • MINUTE(expr)
  • MONTH(expr)
  • SECOND(expr)
  • TIME(expr)
  • YEAR(expr)

Edit this file in your pull request to add your function to the list.

Installation

Just add the package to your composer.json

{
    "require": {
        "luxifer/doctrine-functions": "dev-master"
    }
}

Integration

1) Doctrine Only

According to the Doctrine documentation you can register the functions in this package this way.

<?php
$config = new \Doctrine\ORM\Configuration();
$config->addCustomDatetimeFunction('date', 'Luxifer\DQL\Datetime\Date');

$em = EntityManager::create($dbParams, $config);

2) Using Symfony 2

With symfony 2 you can register yout functions directly in the config.yml file.

doctrine:
    orm:
        entity_managers:
            default:
                dql:
                    datetime_functions:
                        date: Luxifer\DQL\Datetime\Date
